# Lesson 01 — What Makes an App Publish Ready?
## 📖 Explanation
A publish-ready app has no debug code, proper versioning, icons, and passes all policy checks.

## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Try applying this to your own app!
## ⚠️ Common Mistake
Publishing with debug prints still in code.
