# Lesson 03 — Versioning and Build Numbers
## 📖 Explanation
Version is user-facing (1.0.0). Build number is internal (1, 2, 3...).

## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Try applying this to your own app!
## ⚠️ Common Mistake
Forgetting to increment the build number.
