# Lesson 02 — Debug vs Release Builds
## 📖 Explanation
Debug builds include dev tools. Release builds are optimized for users.

## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Try applying this to your own app!
## ⚠️ Common Mistake
Testing only in debug mode before publishing.
