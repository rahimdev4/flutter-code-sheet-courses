# Lesson 17 — Banner Ads
## 📖 Explanation
Banner ads are small rectangles shown at the top or bottom of the screen.

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  google_mobile_ads: ^5.3.0
```

## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Try applying this to your own app!
## ⚠️ Common Mistake
Using real ad IDs during development.
