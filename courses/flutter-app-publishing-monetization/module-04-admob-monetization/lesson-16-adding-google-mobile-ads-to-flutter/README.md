# Lesson 16 — Adding Google Mobile Ads to Flutter
## 📖 Explanation
Add google_mobile_ads package and initialize in main().

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  google_mobile_ads: ^5.3.0
```

## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Try applying this to your own app!
## ⚠️ Common Mistake
Forgetting to call initialize().
