# Lesson 18 — Interstitial Ads
## 📖 Explanation
Interstitial ads are full-screen ads shown between natural transitions.

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  google_mobile_ads: ^5.3.0
```

## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Try applying this to your own app!
## ⚠️ Common Mistake
Showing interstitial on every screen change.
