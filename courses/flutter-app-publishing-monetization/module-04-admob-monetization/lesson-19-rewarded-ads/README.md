# Lesson 19 — Rewarded Ads
## 📖 Explanation
Users watch a video ad and get a reward (coins, hints, premium access).

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  google_mobile_ads: ^5.3.0
```

## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Try applying this to your own app!
## ⚠️ Common Mistake
Granting reward before ad completes.
