# Flutter Developer Roadmap — Checklist

Track your progress through the entire Flutter learning journey. Check off items as you complete them.

---

## Stage 1: Foundation

### Dart Fundamentals
- [ ] Learn variables, data types, and operators
- [ ] Write functions with parameters and return types
- [ ] Use if/else, switch, and loops
- [ ] Work with List, Map, and Set
- [ ] Create classes with constructors and methods
- [ ] Understand inheritance, mixins, and abstract classes
- [ ] Use null safety operators: `?`, `!`, `??`, `?.`
- [ ] Write async/await code with Futures
- [ ] Handle errors with try/catch
- [ ] Practice with enums, extensions, and generics
- [ ] Build a Dart-only mini project

### Flutter Basics
- [ ] Set up Flutter development environment
- [ ] Create your first Flutter app
- [ ] Understand MaterialApp, Scaffold, and AppBar
- [ ] Use Text, Image, Icon, and Button widgets
- [ ] Understand StatelessWidget vs StatefulWidget
- [ ] Use setState for simple state changes
- [ ] Build a counter app from scratch

### Flutter UI and Layouts
- [ ] Use Row, Column, and Stack
- [ ] Use Expanded, Flexible, and SizedBox
- [ ] Create Card and Container with BoxDecoration
- [ ] Build ListView and GridView
- [ ] Use SingleChildScrollView for overflow
- [ ] Apply ThemeData and custom styles
- [ ] Build a responsive layout using MediaQuery
- [ ] Create a profile card, dashboard, and settings screen

### Flutter Navigation and Forms
- [ ] Navigate with Navigator.push and pop
- [ ] Use named routes
- [ ] Pass data between screens
- [ ] Create TextFormField with validation
- [ ] Use GlobalKey<FormState> for form handling
- [ ] Build a complete login/signup flow
