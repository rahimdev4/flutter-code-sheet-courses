# Flutter Developer Roadmap

A complete learning path from zero to professional Flutter developer. This roadmap organizes everything you need to learn — from Dart basics to publishing a real app on the Play Store — into a clear, practical sequence.

---

## Who Is This For?

- Complete beginners who want to become Flutter developers
- Self-taught developers who want a structured path
- Developers switching from other frameworks to Flutter
- Anyone who finished a few tutorials but doesn't know what to learn next

---

## What You Will Understand

After following this roadmap, you will:

- Write clean Dart code with confidence
- Build multi-screen Flutter apps with polished UI
- Manage state professionally with Riverpod
- Connect apps to APIs, Supabase, and Firebase
- Store data locally for offline use
- Structure code using Clean Architecture
- Publish, monetize, and maintain real apps

---

## Roadmap Overview

This roadmap is divided into four stages. Each stage builds on the previous one.

### Stage 1: Foundation (Beginner)

| # | Course | Focus |
|---|--------|-------|
| 01 | [Dart Fundamentals](../dart-fundamentals) | Variables, functions, OOP, async |
| 02 | [Flutter Basics](../flutter-basics) | Widgets, scaffold, hot reload |
| 03 | [Flutter UI & Layouts](../flutter-ui-layouts) | Row, Column, Stack, responsive |
| 04 | [Flutter Navigation & Forms](../flutter-navigation-forms) | Routes, forms, validation |

### Stage 2: App Logic (Intermediate)

| # | Course | Focus |
|---|--------|-------|
| 05 | [Flutter State Management Basics](../flutter-state-management-basics) | setState, lifting state |
| 06 | [Riverpod for Flutter Apps](../riverpod-for-flutter-apps) | Providers, async state, real patterns |

### Stage 3: Data & Backend (Advanced)

| # | Course | Focus |
|---|--------|-------|
| 07 | [Flutter API Integration](../flutter-api-integration) | HTTP, JSON, REST APIs |
| 08 | [Flutter Local Storage & Offline Apps](../flutter-local-storage-offline-apps) | SharedPreferences, Hive, SQLite |
| 09 | [Supabase for Flutter Apps](../supabase-for-flutter-apps) | Auth, database, storage, realtime |
| 10 | [Firebase for Flutter Apps](../firebase-for-flutter-apps) | Auth, Firestore, storage, FCM |

### Stage 4: Professional (Expert)

| # | Course | Focus |
|---|--------|-------|
| 11 | [Flutter Clean Architecture](../flutter-clean-architecture) | Layers, repositories, use cases, DI |
| 12 | [Flutter App Publishing & Monetization](../flutter-app-publishing-monetization) | Play Console, AdMob, ASO, launch |
