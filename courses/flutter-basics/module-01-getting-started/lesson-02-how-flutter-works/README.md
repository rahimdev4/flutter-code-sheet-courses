# Lesson 02 — How Flutter Works

## 📖 Explanation

Flutter uses its own rendering engine to draw every pixel on screen. Unlike other frameworks that use native UI components, Flutter paints everything from scratch — giving you full control over the look and feel of your app.

## 💡 What This Example Demonstrates

- How Flutter renders UI using its own engine
- The widget tree concept
- Hot reload for fast development

## ▶️ How to Run

```bash
flutter run
```

## 🧪 Practice Challenge

Modify the text in the app and observe how hot reload updates the screen instantly without restarting.
