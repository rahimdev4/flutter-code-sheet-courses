# Lesson 04 — Running Your First Flutter App

## 📖 Explanation

Learn how to create and run a Flutter app on an emulator or physical device. This lesson covers `flutter create`, `flutter run`, and debugging basics.

## 💡 What This Example Demonstrates

- A complete minimal Flutter app
- `runApp()` as the app entry point
- Hot reload workflow

## ▶️ How to Run

```bash
flutter run
```

## 🧪 Practice Challenge

Change the app bar color, the body text, and the icon. Use hot reload to see your changes instantly.
