# Lesson 03 — Flutter Project Structure

## 📖 Explanation

Every Flutter project has a standard folder structure. Understanding it helps you know where to put your code, assets, and configuration files.

## 💡 What This Example Demonstrates

- The key files and folders in a Flutter project
- Where `main.dart` lives
- What `pubspec.yaml` does

## ▶️ How to Run

```bash
flutter run
```

## 🧪 Practice Challenge

Create a new Flutter project with `flutter create` and explore each folder. Try adding a new Dart file in the `lib/` folder and importing it.
