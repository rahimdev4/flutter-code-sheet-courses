# 📒 Flutter Basics — Notes

Quick reference notes for the Flutter Basics course.

---

## Key Concepts

- **Flutter** is a UI toolkit by Google for building natively compiled apps.
- Everything in Flutter is a **widget**.
- Flutter uses a **declarative** UI approach.
- The widget tree defines your app's layout.

## Common Commands

```bash
# Create a new Flutter project
flutter create my_app

# Run the app
flutter run

# Build for release
flutter build apk   # Android
flutter build ios    # iOS

# Check Flutter setup
flutter doctor
```
