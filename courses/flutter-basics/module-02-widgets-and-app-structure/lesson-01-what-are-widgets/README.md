# Lesson 01 — What Are Widgets?

## 📖 Explanation

In Flutter, everything is a widget. Widgets describe what the UI should look like. There are two main types: **StatelessWidget** (static UI) and **StatefulWidget** (dynamic UI that changes over time).

## 💡 What This Example Demonstrates

- Creating a `StatelessWidget`
- Creating a `StatefulWidget` with `setState`
- The difference between the two

## ▶️ How to Run

```bash
cd courses/flutter-basics/module-02-widgets/lesson-01-what-are-widgets/
flutter run
```
