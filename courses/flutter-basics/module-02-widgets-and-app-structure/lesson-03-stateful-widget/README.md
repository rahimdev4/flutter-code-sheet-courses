# Lesson 03 — StatefulWidget

## 📖 Explanation

A StatefulWidget can change over time. It holds mutable state that can be updated with `setState()`, causing the widget to rebuild and reflect the new data.

## ▶️ How to Run

```bash
flutter run
```
