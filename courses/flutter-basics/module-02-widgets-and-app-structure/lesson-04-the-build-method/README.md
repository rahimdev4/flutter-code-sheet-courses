# Lesson 04 — The Build Method

## 📖 Explanation

The `build()` method is the heart of every widget. Flutter calls it whenever the widget needs to render. It returns a widget tree that describes the UI.

## ▶️ How to Run

```bash
flutter run
```

## 🧪 Practice Challenge

Create a widget where the build method returns different layouts based on a boolean flag.
