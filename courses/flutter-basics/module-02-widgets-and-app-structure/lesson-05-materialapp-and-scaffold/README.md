# Lesson 05 — MaterialApp and Scaffold

## 📖 Explanation

`MaterialApp` is the root widget that sets up your app's theme, navigation, and Material Design. `Scaffold` provides the basic page structure: AppBar, body, FAB, and more.

## ▶️ How to Run

```bash
flutter run
```

## 🧪 Practice Challenge

Add a `Drawer` and a `BottomNavigationBar` to the Scaffold.
