# Lesson 02 — StatelessWidget

## 📖 Explanation

A StatelessWidget is a widget that never changes once built. It has no mutable state — the same input always produces the same output.

## 💡 What This Example Demonstrates

- Creating a custom StatelessWidget
- Passing data via constructor parameters
- Composing multiple StatelessWidgets

## ▶️ How to Run

```bash
flutter run
```

## 🧪 Practice Challenge

Create a `GreetingCard` StatelessWidget that takes a name and displays a styled greeting.
