# Lesson 02 — Row and Column
## 📖 Explanation
Row arranges children horizontally. Column arranges children vertically. Both support alignment and spacing.
## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Create a layout with a Row of icons inside a Column of text labels.
