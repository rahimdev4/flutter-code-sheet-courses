# 🎯 Flutter Basics

Build your first Flutter apps — from widgets to beautiful UIs.

---

## 👤 Who Is This Course For?

- Developers who have completed the Dart Fundamentals course.
- Anyone who wants to start building mobile apps with Flutter.
- Developers from other frameworks looking to learn Flutter.

## 📚 What You'll Learn

- What Flutter is and how it works
- The widget tree and how Flutter builds UIs
- Core widgets: Text, Container, Row, Column, and more
- How to build complete app screens
