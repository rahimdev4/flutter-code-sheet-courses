# Lesson 03 — Connecting Flutter to Supabase
## 📖 Explanation
Initialize the Supabase client in your main.dart.

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  supabase_flutter: ^2.8.0
```

## ▶️ How to Run
```bash
flutter run
```
