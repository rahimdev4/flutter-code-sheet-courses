# ⚡ Supabase for Flutter Apps

Learn how to build real Flutter apps with Supabase Auth, database CRUD, Row Level Security, Storage, Realtime, clean repository patterns, caching concepts, and a complete notes app project.

*(Note: This course comes after Flutter Local Storage & Offline Apps).*

## ▶️ How to Use Examples

Each lesson folder contains a standalone Flutter example using `supabase_flutter`.

```bash
cd courses/supabase-for-flutter-apps/module-01-supabase-foundations/lesson-01-what-is-supabase/
flutter run
```

> ⚠️ Replace `SUPABASE_URL` and `SUPABASE_ANON_KEY` with your own project credentials.
