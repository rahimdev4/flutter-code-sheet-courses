# Lesson 19 — Uploading Images from Flutter
## 📖 Explanation
Pick an image and upload it to a Supabase storage bucket.

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  supabase_flutter: ^2.8.0
  image_picker: ^1.1.2
```

## ▶️ How to Run
```bash
flutter run
```

Try modifying the example to reinforce your learning!
