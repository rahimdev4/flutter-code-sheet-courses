# Lesson 15 — Boxes, Keys, and Values
## 📖 Explanation
Hive stores data in Boxes. Each Box is like a dictionary of keys and values.

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  hive: ^2.2.3
  hive_flutter: ^1.1.0
```

## ▶️ How to Run
```bash
flutter run
```
