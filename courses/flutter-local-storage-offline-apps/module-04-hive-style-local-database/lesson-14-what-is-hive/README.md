# Lesson 14 — What is Hive?
## 📖 Explanation
Hive is a lightweight, fast NoSQL database for Flutter.

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  hive: ^2.2.3
  hive_flutter: ^1.1.0
```

## ▶️ How to Run
```bash
flutter run
```
