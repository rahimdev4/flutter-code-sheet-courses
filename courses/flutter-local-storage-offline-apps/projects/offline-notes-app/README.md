# 🏗 Final Project: Offline Notes App
A complete app demonstrating local storage with SharedPreferences.

## Features
- Add, delete, and favorite notes
- Data persists across app restarts
- Empty state handling
- Offline-first design

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  shared_preferences: ^2.3.2
```

## ▶️ How to Run
```bash
flutter run
```
