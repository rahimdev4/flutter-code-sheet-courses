# Lesson 18 — Organizing Provider Code
## 📖 Explanation
Use MultiProvider to inject several state classes at the root of your app cleanly.

## ⚠️ Requirements
Ensure you have `provider` in your `pubspec.yaml`:
```yaml
dependencies:
  provider: ^6.1.2
```

## ▶️ How to Run
```bash
flutter run
```
