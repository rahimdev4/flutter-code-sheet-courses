# Lesson 14 — What is Provider?
## 📖 Explanation
Learn how Provider passes data down the widget tree invisibly.

## ⚠️ Requirements
Ensure you have `provider` in your `pubspec.yaml`:
```yaml
dependencies:
  provider: ^6.1.2
```

## ▶️ How to Run
```bash
flutter run
```

Try modifying the example to reinforce your learning!
