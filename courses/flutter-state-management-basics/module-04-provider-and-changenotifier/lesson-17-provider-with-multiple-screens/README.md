# Lesson 17 — Provider with Multiple Screens
## 📖 Explanation
See how Provider easily shares data between entirely separate screens without prop drilling.

## ⚠️ Requirements
Ensure you have `provider` in your `pubspec.yaml`:
```yaml
dependencies:
  provider: ^6.1.2
```

## ▶️ How to Run
```bash
flutter run
```
