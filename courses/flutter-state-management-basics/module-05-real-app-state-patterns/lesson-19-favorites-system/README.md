# Lesson 19 — Favorites System
## 📖 Explanation
Manage a list of favorite IDs in state and conditionally display active icons.

## ⚠️ Requirements
Ensure you have `provider` in your `pubspec.yaml`:
```yaml
dependencies:
  provider: ^6.1.2
```

## ▶️ How to Run
```bash
flutter run
```
