# 🏗 Final Project: Notes State App
A complete app demonstrating State Management with Provider.

## Features
- View a list of notes
- Add new notes
- Mark notes as favorites
- Empty state handling

## ⚠️ Requirements
Ensure you have `provider` in your `pubspec.yaml`:
```yaml
dependencies:
  provider: ^6.1.2
```

## ▶️ How to Run
```bash
flutter run
```
