# 🔄 Flutter State Management Basics

Learn how Flutter apps remember values, update screens, manage shared data, and build real app features using `setState`, `ValueNotifier`, and `Provider`.
*(Note: This course teaches core state concepts before diving into advanced solutions like Riverpod).*

## ▶️ How to Use Examples

Each lesson folder contains a standalone Flutter example:

```bash
cd courses/flutter-state-management-basics/module-01-state-management-foundations/lesson-01-what-is-state-in-flutter/
flutter run
```

*(For Provider lessons, ensure your project `pubspec.yaml` includes `provider: ^6.1.2`)*
