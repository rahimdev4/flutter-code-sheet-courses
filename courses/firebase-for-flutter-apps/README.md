# 🔥 Firebase for Flutter Apps

Learn how to build real Flutter apps with Firebase Auth, Firestore, Storage, Security Rules, Cloud Messaging concepts, Analytics concepts, and a complete Firebase Notes App project.

*(Note: This course comes after Supabase for Flutter Apps).*

## ▶️ How to Use Examples

Each lesson folder contains a standalone Flutter example.

```bash
cd courses/firebase-for-flutter-apps/module-01-firebase-foundations/lesson-01-what-is-firebase/
flutter run
```

> ⚠️ You must configure Firebase via FlutterFire CLI before running examples that use Firebase packages.
