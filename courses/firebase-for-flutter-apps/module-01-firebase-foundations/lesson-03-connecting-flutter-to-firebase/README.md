# Lesson 03 — Connecting Flutter to Firebase
## 📖 Explanation
Use FlutterFire CLI to generate firebase_options.dart.

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  firebase_core: ^3.6.0
```

## ▶️ How to Run
```bash
flutter run
```
