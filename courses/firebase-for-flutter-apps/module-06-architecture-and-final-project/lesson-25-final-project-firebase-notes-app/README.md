# Lesson 25 — Final Project: Firebase Notes App
## 📖 Explanation
Put everything together: Auth, Firestore CRUD, Security Rules, Repository, Riverpod.

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  firebase_core: ^3.6.0
  firebase_auth: ^5.3.1
  cloud_firestore: ^5.4.4
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
