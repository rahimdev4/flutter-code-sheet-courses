# Lesson 19 — Uploading Images from Flutter
## 📖 Explanation
Pick an image and upload it to Firebase Storage.

## ⚠️ Requirements
```yaml
dependencies:
  flutter:
    sdk: flutter
  firebase_core: ^3.6.0
  firebase_storage: ^12.3.4
  image_picker: ^1.1.2
```

## ▶️ How to Run
```bash
flutter run
```
