# Lesson 09 — Using Consumer
## 📖 Explanation
Use a Consumer builder to rebuild only a specific part of a widget tree.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```

Try modifying the example to reinforce your learning!
