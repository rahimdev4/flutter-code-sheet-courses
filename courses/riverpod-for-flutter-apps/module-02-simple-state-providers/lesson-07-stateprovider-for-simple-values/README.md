# Lesson 07 — StateProvider for Simple Values
## 📖 Explanation
Use StateProvider to manage basic mutable state like ints, booleans, or strings.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
