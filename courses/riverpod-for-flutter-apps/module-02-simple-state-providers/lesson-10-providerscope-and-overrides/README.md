# Lesson 10 — ProviderScope and Overrides
## 📖 Explanation
Override providers for testing or scoping data to a specific widget subtree.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
