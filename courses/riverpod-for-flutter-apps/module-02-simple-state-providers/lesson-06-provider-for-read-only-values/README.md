# Lesson 06 — Provider for Read Only Values
## 📖 Explanation
Use the base Provider class for values that never change or for dependency injection.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
