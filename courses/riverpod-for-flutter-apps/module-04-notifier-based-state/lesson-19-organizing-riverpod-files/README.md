# Lesson 19 — Organizing Riverpod Files
## 📖 Explanation
Best practices: Keep your provider definitions right next to your logic classes.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
