# Lesson 16 — Creating a Counter Notifier
## 📖 Explanation
Build a robust counter using Notifier instead of StateProvider.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
