# Lesson 17 — StateNotifier vs Notifier
## 📖 Explanation
Understand why Notifier is better (you can access the 'ref' inside the class!).

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
