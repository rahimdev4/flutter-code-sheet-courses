# Lesson 03 — Setting Up Riverpod
## 📖 Explanation
Wrap your app in a ProviderScope to initialize Riverpod.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
