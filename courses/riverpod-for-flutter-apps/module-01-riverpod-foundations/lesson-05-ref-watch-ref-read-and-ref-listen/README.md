# Lesson 05 — ref.watch, ref.read, and ref.listen
## 📖 Explanation
Understand when to use watch (in build), read (in callbacks), and listen (for side effects).

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```

Try modifying the example to reinforce your learning!
