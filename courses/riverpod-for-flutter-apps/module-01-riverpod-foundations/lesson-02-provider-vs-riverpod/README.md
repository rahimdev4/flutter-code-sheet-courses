# Lesson 02 — Provider vs Riverpod
## 📖 Explanation
Understand why Riverpod uses global variables instead of the widget tree.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```

Try modifying the example to reinforce your learning!
