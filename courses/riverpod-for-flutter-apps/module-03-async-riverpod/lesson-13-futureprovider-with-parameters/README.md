# Lesson 13 — FutureProvider with Parameters
## 📖 Explanation
Use the .family modifier to pass arguments to your FutureProvider.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
