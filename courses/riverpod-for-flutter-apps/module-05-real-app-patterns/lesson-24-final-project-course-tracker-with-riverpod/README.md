# Lesson 24 — Final Project: Course Tracker with Riverpod
## 📖 Explanation
Put everything together: Providers, Notifiers, StateProviders, and UI logic.

Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml

  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
