# Lesson 21 — Theme Switcher with Riverpod
## 📖 Explanation
Toggle dark and light mode effortlessly using StateProvider.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
