# Lesson 20 — Favorites System with Riverpod
## 📖 Explanation
Manage a list of favorite items using a Notifier.

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
