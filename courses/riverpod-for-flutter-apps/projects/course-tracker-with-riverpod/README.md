# 🏗 Final Project: Course Tracker with Riverpod
A complete app demonstrating Riverpod State Management.

## Features
- List of courses (Async Data)
- Mark courses as favorite
- Toggle dark/light theme
- Overall progress calculation

## ⚠️ Requirements
Ensure you have `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
