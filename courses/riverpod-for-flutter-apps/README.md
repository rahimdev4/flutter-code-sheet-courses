# 🌊 Riverpod for Flutter Apps

Learn modern Flutter state management with **Riverpod**. This course covers Provider, StateProvider, ConsumerWidget, async providers, Notifier, AsyncNotifier, and real app patterns.

*(Note: This course comes after Flutter State Management Basics).*

## ▶️ How to Use Examples

Each lesson folder contains a standalone Flutter example using `flutter_riverpod`.

```bash
cd courses/riverpod-for-flutter-apps/module-01-riverpod-foundations/lesson-01-what-is-riverpod/
flutter run
```

*(Ensure your project `pubspec.yaml` includes `flutter_riverpod: ^2.6.1`)*
