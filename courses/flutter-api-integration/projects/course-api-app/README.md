# 🏗 Final Project: Course API App
A complete app demonstrating HTTP API requests, JSON parsing, models, Riverpod state, searching, and pull-to-refresh.

## Features
- Fetches a list of mocked courses via `http`
- Pull to Refresh
- Search Filtering
- Riverpod for State Management

## ⚠️ Requirements
Ensure you have `http` and `flutter_riverpod` in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  http: ^1.2.2
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
