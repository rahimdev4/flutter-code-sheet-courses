# Lesson 10 — Showing API Data in Flutter UI
## 📖 Explanation
Use FutureBuilder for simple one-off UI fetching without State.

## ⚠️ Requirements
Ensure you have the required packages in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  http: ^1.2.2
```

## ▶️ How to Run
```bash
flutter run
```
