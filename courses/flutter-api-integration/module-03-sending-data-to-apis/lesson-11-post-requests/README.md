# Lesson 11 — POST Requests
## 📖 Explanation
Send JSON data to create a new resource on the server.

## ⚠️ Requirements
Ensure you have the required packages in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  http: ^1.2.2
```

## ▶️ How to Run
```bash
flutter run
```
