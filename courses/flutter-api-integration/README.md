# 🌐 Flutter API Integration

Learn how to connect Flutter apps with real backend data using HTTP requests, JSON parsing, models, error handling, pagination, search, authentication concepts, and Riverpod based API state.

*(Note: This course comes after Riverpod for Flutter Apps).*

## ▶️ How to Use Examples

Each lesson folder contains a standalone Flutter example using `http`.

```bash
cd courses/flutter-api-integration/module-01-api-foundations/lesson-01-what-is-api-integration/
flutter run
```

*(Ensure your project `pubspec.yaml` includes `http: ^1.2.2`)*
