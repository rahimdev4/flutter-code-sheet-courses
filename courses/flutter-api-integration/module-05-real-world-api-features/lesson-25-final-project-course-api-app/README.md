# Lesson 25 — Final Project: Course API App
## 📖 Explanation
Put everything together using the HTTP package and Riverpod.

## ⚠️ Requirements
Ensure you have the required packages in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  http: ^1.2.2
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
