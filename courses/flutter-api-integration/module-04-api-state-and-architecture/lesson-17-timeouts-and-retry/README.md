# Lesson 17 — Timeouts and Retry
## 📖 Explanation
Don't let your app hang forever if the network is bad.

## ⚠️ Requirements
Ensure you have the required packages in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  http: ^1.2.2
```

## ▶️ How to Run
```bash
flutter run
```
