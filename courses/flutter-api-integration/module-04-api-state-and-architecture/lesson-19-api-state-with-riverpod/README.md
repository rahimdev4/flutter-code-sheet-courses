# Lesson 19 — API State with Riverpod
## 📖 Explanation
Combine Service pattern with Riverpod FutureProvider.

## ⚠️ Requirements
Ensure you have the required packages in your `pubspec.yaml`:
```yaml
dependencies:
  flutter:
    sdk: flutter
  http: ^1.2.2
  flutter_riverpod: ^2.6.1
```

## ▶️ How to Run
```bash
flutter run
```
