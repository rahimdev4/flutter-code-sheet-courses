# Lesson 23 — Empty, Loading, and Error States
## 📖 Explanation
Always handle all three states in your UI: empty (no data), loading (fetching), and error (something went wrong).
## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Add a retry button to the error state that reloads data.
