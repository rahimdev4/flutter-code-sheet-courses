# 🎨 Flutter UI & Layouts

## ▶️ How to Use Examples

Each lesson folder contains a standalone Flutter example:

```bash
cd courses/flutter-ui-layouts/module-01-layout-thinking/lesson-01-ui-design-mindset-for-flutter/
flutter run
```

Or paste the code into your own Flutter project's `lib/main.dart`.
