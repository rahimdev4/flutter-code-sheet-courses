# 📦 Module 03 — Scrollable Interfaces

## 🎯 Module Goal
Learn to build scrollable UIs — from simple scroll views to dynamic lists, grids, and paged views.

## 📚 Lessons
| # | Lesson | Description |
|---|--------|-------------|
| 10 | [SingleChildScrollView](lesson-10-singlechildscrollview/) | Make any content scrollable |
| 11 | [ListView Basics](lesson-11-listview-basics/) | Dynamic scrollable lists |
| 12 | [GridView Basics](lesson-12-gridview-basics/) | Grid layouts |
| 13 | [List Tiles and Sectioned Lists](lesson-13-list-tiles-and-sectioned-lists/) | Structured list items |
| 14 | [PageView for Onboarding](lesson-14-pageview-for-onboarding/) | Swipeable pages |

## ✅ After This Module You Should Know
- When to use SingleChildScrollView vs ListView
- How to build dynamic lists with ListView.builder
- How to create grid layouts
- How to build sectioned lists with headers
- How to create page-based UIs
