# 🏗 Learning Dashboard UI — Final Project

A complete learning dashboard that combines all concepts from the Flutter UI & Layouts course.

## Features
- Custom header with avatar and greeting
- Stats row with reusable StatCard widgets
- Course progress list with progress bars
- Horizontal scrolling explore section
- Bottom navigation bar

## ▶️ How to Run
```bash
flutter run
```

## 🎨 Design
Uses the course color palette:
- Background: `#050F19`
- Primary: `#00D9FF`
- Secondary: `#00C896`
- Cards: `#0B1A2A`
