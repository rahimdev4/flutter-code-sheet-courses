# Lesson 01 — UI Design Mindset for Flutter
## 📖 Explanation
In Flutter, everything is a widget. Learning to think in widgets means breaking any screen into small, composable pieces.
## 💡 What This Example Demonstrates
- Breaking a UI into widget components
- Composing widgets to form a complete screen
## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Take any app screenshot and break it down into a list of Flutter widgets you'd use.
