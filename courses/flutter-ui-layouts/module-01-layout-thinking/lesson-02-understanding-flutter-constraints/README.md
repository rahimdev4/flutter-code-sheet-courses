# Lesson 02 — Understanding Flutter Constraints
## 📖 Explanation
Flutter uses a constraint-based layout: parents pass constraints down, children pick a size, parents position the child.
## 💡 What This Example Demonstrates
- How tight vs loose constraints work
- How `Container` behaves with/without size
## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Wrap a Container inside another Container with different constraints and observe the behavior.
