# Lesson 07 — Row and Column
## 📖 Explanation
Row lays out children horizontally. Column lays out children vertically. Use mainAxisAlignment and crossAxisAlignment to control positioning.
## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Create a stats bar with 3 items in a Row, each containing a Column with an icon and label.
