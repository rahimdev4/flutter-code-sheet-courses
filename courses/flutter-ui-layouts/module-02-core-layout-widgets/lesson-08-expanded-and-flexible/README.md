# Lesson 08 — Expanded and Flexible
## 📖 Explanation
Expanded forces a child to fill all remaining space. Flexible lets a child take up space proportionally using flex values.
## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Create a layout with 3 boxes using flex values 1, 2, and 1.
