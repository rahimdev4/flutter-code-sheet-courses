# Lesson 06 — Padding and Margin
## 📖 Explanation
Padding adds space inside a widget. Margin (via Container) adds space outside. Use Padding widget or Container's padding/margin.
## ▶️ How to Run
```bash
flutter run
```
## 🧪 Practice Challenge
Create two boxes side by side — one using only Padding, one using only margin.
