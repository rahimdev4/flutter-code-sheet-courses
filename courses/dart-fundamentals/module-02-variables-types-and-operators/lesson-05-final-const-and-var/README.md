# Lesson 05 — Final, Const, and Var

## 📖 Explanation

Dart gives you three ways to declare variables: `var` for mutable values, `final` for values set once at runtime, and `const` for compile-time constants. Choosing the right one makes your code safer and clearer.

## 💡 What This Example Demonstrates

- `var` — mutable, can be reassigned
- `final` — set once at runtime, cannot be reassigned
- `const` — compile-time constant, must be known at compile time
- When to use each one

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create a program with: a `var` for a counter that increments, a `final` for the current date/time, and a `const` for the value of pi. Print all three.
