# Lesson 04 — Variables and Values

## 📖 Explanation

Variables are named containers that store values. In Dart, every variable has a type. You can declare them explicitly or let Dart figure out the type automatically.

## 💡 What This Example Demonstrates

- Declaring variables with explicit types (`String`, `int`, `double`, `bool`)
- Type inference with `var`
- Storing and updating values
- Understanding Dart's type system

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create variables to store a movie's title, release year, rating (out of 10), and whether you've watched it. Print them all.
