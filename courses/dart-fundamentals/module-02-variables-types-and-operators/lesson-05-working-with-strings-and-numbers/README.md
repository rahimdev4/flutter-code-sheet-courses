# Lesson 05 — Working with Strings and Numbers

## 📖 Explanation

Strings and numbers are the most commonly used data types. Dart provides rich methods for manipulating text and performing math operations.

## 💡 What This Example Demonstrates

- String methods: `toUpperCase()`, `contains()`, `split()`
- String interpolation with `$` and `${}`
- Number operations and type conversion

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create a program that takes a full name, splits it into first and last name, and prints each in uppercase.
