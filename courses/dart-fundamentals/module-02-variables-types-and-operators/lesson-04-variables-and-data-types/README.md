# Lesson 04 — Variables and Data Types

## 📖 Explanation

Variables are containers for storing data. Dart is a statically typed language, meaning every variable has a type. You can declare variables with explicit types or let Dart infer them.

## 💡 What This Example Demonstrates

- Declaring variables with `String`, `int`, `double`, `bool`
- Using `var` for type inference
- Using `final` and `const` for immutable values

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create variables to store a book's title, page count, price, and whether you've read it. Print them all.
