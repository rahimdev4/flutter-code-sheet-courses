# Lesson 06 — Operators in Dart

## 📖 Explanation

Operators are symbols that perform operations on values. Dart supports arithmetic, comparison, logical, and assignment operators.

## 💡 What This Example Demonstrates

- Arithmetic operators: `+`, `-`, `*`, `/`, `~/`, `%`
- Comparison operators: `==`, `!=`, `>`, `<`, `>=`, `<=`
- Logical operators: `&&`, `||`, `!`

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Write a program that checks if a number is positive, negative, or zero using comparison operators.
