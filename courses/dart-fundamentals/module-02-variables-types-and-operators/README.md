# 📦 Module 02 — Variables, Types, and Operators

Learn how Dart stores and manipulates data.

---

## 🎯 Module Goal

Understand Dart's type system, the difference between `var`, `final`, and `const`, and how to use operators.

## 📚 Lessons

| # | Lesson | Description |
|---|--------|-------------|
| 4 | [Variables and Values](lesson-04-variables-and-values/) | Declaring and using variables |
| 5 | [Final, Const, and Var](lesson-05-final-const-and-var/) | Choosing the right declaration |
| 6 | [Operators in Dart](lesson-06-operators-in-dart/) | Arithmetic, comparison, and logical operators |

## ✅ After This Module You Should Know

- How to declare variables with explicit types and type inference
- When to use `var`, `final`, and `const`
- How to use arithmetic, comparison, and logical operators
