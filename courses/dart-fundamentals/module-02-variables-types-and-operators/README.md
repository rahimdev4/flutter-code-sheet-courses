# 📦 Module 02 — Variables, Types & Operators

Learn how Dart stores and manipulates data.

---

## 🎯 Module Goal

Understand Dart's type system, work with strings and numbers, and use operators to perform calculations and comparisons.

## 📚 Lessons

| # | Lesson | Description |
|---|--------|-------------|
| 4 | [Variables and Data Types](lesson-04-variables-and-data-types/) | Declaring and using variables |
| 5 | [Working with Strings and Numbers](lesson-05-working-with-strings-and-numbers/) | String methods and number operations |
| 6 | [Operators in Dart](lesson-06-operators-in-dart/) | Arithmetic, comparison, and logical operators |

## ✅ After This Module You Should Know

- How to declare variables using `var`, `final`, `const`, and explicit types
- Dart's built-in data types: `String`, `int`, `double`, `bool`
- How to manipulate strings and perform arithmetic
- How to use comparison and logical operators
