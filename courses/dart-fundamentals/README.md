# 🎯 Dart Fundamentals

Master the Dart programming language from scratch — the foundation of every Flutter app.

---

## 👤 Who Is This Course For?

- Complete beginners with no prior programming experience.
- Developers from other languages who want to learn Dart quickly.
- Anyone preparing to learn Flutter.

## 📚 What You'll Learn

- Dart syntax, variables, and data types
- Control flow (if/else, switch, loops)
- Functions and collections (Lists, Maps, Sets)
- Object-oriented programming (classes, constructors, inheritance, JSON)
- Modern Dart features (null safety, async/await, error handling)
