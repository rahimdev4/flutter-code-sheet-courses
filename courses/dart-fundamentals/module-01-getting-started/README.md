# 📦 Module 01 — Getting Started with Dart

Your first steps with the Dart programming language.

---

## 🎯 Module Goal

By the end of this module, you will understand what Dart is, how to set up your environment, and how Dart code executes.

- What Dart is and why it's used with Flutter

- How Dart compiles and executes your code
