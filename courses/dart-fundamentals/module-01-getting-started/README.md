# 📦 Module 01 — Getting Started

Your first steps with the Dart programming language.

---

## 🎯 Module Goal

By the end of this module, you will understand what Dart is, how to set up your environment, and how to write and run your first Dart program.

## 📚 Lessons

| # | Lesson | Description |
|---|--------|-------------|
| 1 | [What Is Dart?](lesson-01-what-is-dart/) | Introduction to the Dart language |
| 2 | [DartPad and Setup](lesson-02-dartpad-and-setup/) | Setting up your development environment |
| 3 | [Your First Dart Program](lesson-03-your-first-dart-program/) | Writing and running your first program |

## ✅ After This Module You Should Know

- What Dart is and why it's used with Flutter
- How to use DartPad or a local IDE to write Dart code
- How to write, run, and understand a basic Dart program
