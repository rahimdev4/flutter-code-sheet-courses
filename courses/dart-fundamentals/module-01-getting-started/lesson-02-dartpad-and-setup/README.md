# Lesson 02 — DartPad and Setup

## 📖 Explanation

Before writing Dart code, you need a place to run it. The easiest option is **DartPad** — an online editor that runs Dart in your browser. For local development, you can install the Dart SDK or use VS Code with the Dart extension.

## 💡 What This Example Demonstrates

- Running a Dart program that confirms your setup is working
- Using string interpolation with `$variable`

## ▶️ How to Run

```bash
dart run main.dart
```

Or paste the code into [DartPad](https://dartpad.dev).

## 🧪 Practice Challenge

Add a variable for your favorite IDE and print it.
