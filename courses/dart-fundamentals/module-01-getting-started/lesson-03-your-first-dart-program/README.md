# Lesson 03 — Your First Dart Program

## 📖 Explanation

In this lesson, you'll write a complete Dart program from scratch. You'll learn about the `main()` function, variables, and how to display output using `print()`.

## 💡 What This Example Demonstrates

- Declaring variables with types
- String interpolation
- Printing formatted output

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create a program that prints a short introduction about yourself — your name, age, and what you want to build with Flutter.
