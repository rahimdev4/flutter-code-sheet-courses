# Lesson 01 — What Is Dart?

## 📖 Explanation

Dart is an open-source, general-purpose programming language developed by Google. It is the language used to build Flutter apps for mobile, web, and desktop from a single codebase.

## 💡 What This Example Demonstrates

- The basic structure of a Dart program
- The `main()` function as the entry point
- Using `print()` to display output

## ▶️ How to Run

```bash
dart run main.dart
```

Or paste the code into [DartPad](https://dartpad.dev).

## 🧪 Practice Challenge

Modify the program to print your name and your favorite programming language.
