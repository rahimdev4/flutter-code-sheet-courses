# Lesson 03 — How Dart Code Runs

## 📖 Explanation

Understanding how Dart code runs helps you write better programs. Dart code is executed line by line, starting from the `main()` function. Dart can run in two modes: JIT (Just-In-Time) for development and AOT (Ahead-Of-Time) for production.

## 💡 What This Example Demonstrates

- The execution flow of a Dart program
- How code runs sequentially from top to bottom
- The difference between compile-time and runtime
- Basic program structure and flow

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Add more print statements at different points in the program to observe the exact order of execution. Try calling a function before and after defining it.
