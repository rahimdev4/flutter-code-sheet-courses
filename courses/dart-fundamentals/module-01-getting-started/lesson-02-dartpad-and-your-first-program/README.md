# Lesson 02 — DartPad and Your First Program

## 📖 Explanation

DartPad is a free online editor that lets you write and run Dart code directly in your browser — no installation needed. In this lesson, you'll set up your coding environment and write your very first Dart program.

## 💡 What This Example Demonstrates

- How to use DartPad to write Dart code
- The `main()` function as the entry point of every Dart program
- Using `print()` to display output
- Declaring and using your first variables

## ▶️ How to Run

```bash
dart run main.dart
```

Or paste the code into [DartPad](https://dartpad.dev).

## 🧪 Practice Challenge

Write a program that prints your name, your age, and your favorite programming language — each on a separate line.
