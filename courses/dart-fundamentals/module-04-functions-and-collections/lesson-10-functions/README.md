# Lesson 10 — Functions

## 📖 Explanation

Functions are reusable blocks of code that perform a specific task. They help keep your code organized and avoid repetition.

## 💡 What This Example Demonstrates

- Defining functions with return types
- Positional and named parameters
- Default parameter values
- Arrow functions (short syntax)

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Write a function called `isEven` that takes an integer and returns `true` if it's even, `false` otherwise. Test it with several numbers.
