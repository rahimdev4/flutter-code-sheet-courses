# Lesson 11 — Lists

## 📖 Explanation

A List is an ordered collection of items. Lists are one of the most commonly used data structures in Dart, used to store multiple values of the same type.

## 💡 What This Example Demonstrates

- Creating lists
- Adding, removing, and accessing elements
- Iterating over lists
- Useful list methods

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create a list of 5 cities. Sort them alphabetically, then print each city with its index number.
