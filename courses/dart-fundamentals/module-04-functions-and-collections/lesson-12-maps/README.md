# Lesson 12 — Maps

## 📖 Explanation

A Map is a collection of key-value pairs. Each key maps to exactly one value. Maps are useful for storing structured data like user profiles, settings, or lookup tables.

## 💡 What This Example Demonstrates

- Creating maps
- Adding, accessing, and removing entries
- Iterating over keys and values
- Checking if a key or value exists

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create a Map that stores country names as keys and their capitals as values. Print each pair in the format: "The capital of X is Y."
