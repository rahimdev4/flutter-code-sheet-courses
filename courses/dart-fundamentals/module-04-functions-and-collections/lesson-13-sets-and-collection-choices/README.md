# Lesson 13 — Sets and Collection Choices

## 📖 Explanation

A Set is an unordered collection of unique items — no duplicates allowed. Knowing when to use a List, Set, or Map is an important skill for writing clean Dart code.

## 💡 What This Example Demonstrates

- Creating and using Sets
- Adding and removing elements
- Set operations: union, intersection, difference
- Choosing between List, Set, and Map

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create two sets of student names from two different classes. Find students who are in both classes, and students who are only in one class.
