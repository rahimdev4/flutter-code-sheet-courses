# 📒 Dart Fundamentals — Notes

Quick reference notes for the Dart Fundamentals course.

---

## Key Concepts

- **Dart** is an open-source, general-purpose programming language by Google.
- Dart is the language behind **Flutter**.
- Dart supports **strong typing**, **null safety**, and **async/await**.

## Common Commands

```bash
# Run a Dart file
dart run main.dart

# Check Dart version
dart --version

# Analyze code for issues
dart analyze
```
