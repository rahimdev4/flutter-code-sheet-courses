# Lesson 14 — Constructors

## 📖 Explanation

Constructors are special methods that initialize an object when it's created. Dart supports several types of constructors for different scenarios.

## 💡 What This Example Demonstrates

- Default constructor
- Short-hand constructor with `this.`
- Named constructors
- Factory constructors

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create a `Color` class with `r`, `g`, `b` properties. Add a named constructor `Color.white()` that sets all values to 255 and `Color.black()` that sets all to 0.
