# Lesson 15 — Inheritance

## 📖 Explanation

Inheritance allows a class to inherit properties and methods from another class. The child class can extend the parent class and add its own features or override existing behavior.

## 💡 What This Example Demonstrates

- Extending a parent class with `extends`
- Using `super` to call the parent constructor
- Overriding methods with `@override`

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create a `Vehicle` class with `brand` and `speed` properties. Then create `Car` and `Bicycle` subclasses that override a `describe()` method.
