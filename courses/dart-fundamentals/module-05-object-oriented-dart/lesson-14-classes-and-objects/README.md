# Lesson 13 — Classes and Objects

## 📖 Explanation

A class is a blueprint for creating objects. Objects are instances of a class that hold data (properties) and behavior (methods). OOP helps you organize code into logical, reusable structures.

## 💡 What This Example Demonstrates

- Defining a class with properties and methods
- Creating objects from a class
- Accessing properties and calling methods

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create a `Book` class with properties for `title`, `author`, and `pages`. Add a method that prints a summary of the book.
