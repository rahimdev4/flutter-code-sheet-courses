# 📦 Module 05 — Object Oriented Dart

Build structured programs with classes and objects.

---

## 🎯 Module Goal

Learn the fundamentals of object-oriented programming (OOP) in Dart: classes, constructors, inheritance, and model classes.

## 📚 Lessons

| # | Lesson | Description |
|---|--------|-------------|
| 14 | [Classes and Objects](lesson-14-classes-and-objects/) | Creating your own types |
| 15 | [Constructors](lesson-15-constructors/) | Initializing objects |
| 16 | [Inheritance](lesson-16-inheritance/) | Extending classes |
| 17 | [Model Classes and JSON](lesson-17-model-classes-and-json/) | Converting between JSON and Dart objects |

## ✅ After This Module You Should Know

- How to define classes with properties and methods
- Different types of constructors in Dart
- How inheritance lets you reuse and extend code
- How to create model classes with fromJson/toJson
