# 📦 Module 05 — Object-Oriented Dart

Build structured programs with classes and objects.

---

## 🎯 Module Goal

Learn the fundamentals of object-oriented programming (OOP) in Dart: classes, objects, constructors, and inheritance.

| # | Lesson | Description |

## ✅ After This Module You Should Know

- How to define classes with properties and methods
- How to create objects (instances) from a class
- Different types of constructors in Dart
- How inheritance lets you reuse and extend code
