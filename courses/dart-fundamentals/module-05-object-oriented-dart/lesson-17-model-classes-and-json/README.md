# Lesson 17 — Model Classes and JSON

## 📖 Explanation

In real apps, you often receive data as JSON (from APIs or databases). Model classes let you represent that data as Dart objects. Learning to convert between JSON and Dart objects (`fromJson`/`toJson`) is essential for Flutter development.

## 💡 What This Example Demonstrates

- Creating a model class with properties
- Writing a `fromJson` factory constructor
- Writing a `toJson` method
- Converting between JSON maps and Dart objects

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Create a `Product` model class with `name`, `price`, and `inStock` fields. Add `fromJson` and `toJson` methods. Create a list of products from JSON data and print them.
