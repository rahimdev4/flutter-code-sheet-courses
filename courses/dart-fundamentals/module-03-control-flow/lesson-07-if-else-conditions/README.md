# Lesson 07 — If/Else Conditions

## 📖 Explanation

Conditions let your program make decisions. The `if` statement runs code only when a condition is true. Use `else if` for additional checks and `else` as a fallback.

## 💡 What This Example Demonstrates

- Basic `if/else` structure
- `else if` for multiple conditions
- Nested conditions

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Write a program that takes a test score and prints the grade: A (90+), B (80+), C (70+), D (60+), F (below 60).
