# Lesson 08 — Switch Statements

## 📖 Explanation

The `switch` statement is a cleaner way to compare a single value against multiple options. It's often used instead of long `if/else if` chains.

## 💡 What This Example Demonstrates

- Basic `switch` with `case` and `default`
- Using `break` to stop execution
- Matching string values

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Write a switch statement that prints the number of days in a given month name.
