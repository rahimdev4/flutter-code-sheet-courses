# Lesson 20 — Try, Catch, and Exceptions

## 📖 Explanation

Errors happen — files might not exist, network calls can fail, or users might enter invalid data. Dart's `try/catch` lets you handle errors gracefully instead of crashing your app.

## 💡 What This Example Demonstrates

- Using `try/catch` to handle errors
- The `finally` block for cleanup code
- Throwing custom exceptions
- Catching specific exception types

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Write a function `divide(int a, int b)` that throws an exception if `b` is zero. Use try/catch to handle the error and print a friendly message.
