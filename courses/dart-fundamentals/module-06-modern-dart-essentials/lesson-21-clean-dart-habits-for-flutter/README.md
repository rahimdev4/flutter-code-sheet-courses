# Lesson 21 — Clean Dart Habits for Flutter

## 📖 Explanation

Before jumping into Flutter, it's important to develop clean coding habits. This lesson covers best practices that will make your Flutter code easier to read, maintain, and debug.

## 💡 What This Example Demonstrates

- Meaningful variable and function names
- Using `final` and `const` by default
- Writing short, focused functions
- Proper use of comments
- Avoiding common beginner mistakes

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Refactor a messy piece of code: rename vague variables, break a long function into smaller ones, and add clear comments. Practice writing code that reads like English.
