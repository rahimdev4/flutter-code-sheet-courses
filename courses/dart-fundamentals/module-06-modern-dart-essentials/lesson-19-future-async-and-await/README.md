# Lesson 19 — Future, Async, and Await

## 📖 Explanation

Asynchronous programming lets your program do other work while waiting for slow operations (like network requests or file reads). Dart uses `Future`, `async`, and `await` to handle async code cleanly.

## 💡 What This Example Demonstrates

- Creating a `Future` that completes after a delay
- Using `async` and `await` to write clean async code
- Running multiple futures

## ▶️ How to Run

```bash
dart run main.dart
```

## 🧪 Practice Challenge

Write an async function that simulates checking a username's availability (with a 1-second delay). Return `true` if the name is "admin", `false` otherwise.
