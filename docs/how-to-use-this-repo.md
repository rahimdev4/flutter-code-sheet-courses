# 🎓 How to Use This Repo

A quick guide for students using this repository alongside the Flutter Code Sheet app.

---

## Step 1: Clone the Repo

```bash
git clone https://github.com/your-username/flutter-code-sheet-courses.git
cd flutter-code-sheet-courses
```

## Step 2: Find Your Lesson

Browse to the course and lesson you're working on:

```
courses/dart-fundamentals/module-01-getting-started/lesson-01-what-is-dart/
```

Each lesson folder contains:
- **README.md** — Explanation of the topic and a practice challenge.
- **main.dart** (or `lib/main.dart`) — Working code example.

## Step 3: Run the Code

### For Dart Lessons

```bash
dart run main.dart
```

### For Flutter Lessons

```bash
cd courses/flutter-basics/module-01-getting-started/lesson-01-what-is-flutter/
flutter run
```

### Using DartPad (No Setup Required)

1. Open [DartPad](https://dartpad.dev).
2. Copy the code from `main.dart`.
3. Paste and click **Run**.

## Step 4: Try the Practice Challenge

Each lesson README ends with a practice challenge. Try to solve it on your own before looking at the next lesson!

## Step 5: Track Your Progress

Open the **Flutter Code Sheet** app to mark lessons as complete, take quizzes, and track your learning streak.
